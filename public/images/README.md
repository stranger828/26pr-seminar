# Image Asset Guide

## Folder roles

- `brand/`: logo, seminar title, brand lockup
- `heroes/`: top banner visuals, title + character combinations
- `characters/`: standalone character images and variations
- `tasks/`: task-specific cover images
- `icons/`: small icons and simple symbols

## Naming rule

Use lowercase kebab-case.

Pattern:

`[screen-or-role]-[content]-[variant].[ext]`

Examples:

- `home-hero-title-character.webp`
- `home-hero-title-character-mobile.webp`
- `brand-seminar-logo.svg`
- `character-guide-wave.png`
- `task-writing-cover.png`

## First recommended files

- `heroes/home-hero-title-character.webp`
- `brand/brand-seminar-logo.svg`
